# Unfair Flips - Archipelago Mod

## Install Guide
1. Download [BepinEx](https://github.com/BepInEx/BepInEx/releases) 5.4 for x64 architecture
2. Extract all files into your Unfair Flips directory
3. Launch the game
4. Close the game
5. Download the [latest release of this mod](https://github.com/robotzurg/UnfairFlipsAPMod/releases)
6. Extract the UnfairFlipsAPMod folder into the bepinex/plugins folder in your game folder
7. Launch the game
8. Flip the coin
9. Land on tails
10. Cry

## How do ranomize work?
Flip can only be heads if you have enough fairness.
Money can only be big if you have enough fairness.
